    Style Bundle

    Style bundle is a predefined CSS file that styles a survey in a specific way. Each responsive layout theme has an option to select a predefined style bundle. When a style bundle is selected, it is possible to customize it via a number of variables. For example you can set the image URL to be displayed in the survey header, or change the colors used in the survey. Additional customization can be performed on the Custom CSS tab (for example overriding or adding new styles). It is also possible to select no style bundle, and create a completely custom styling for a survey on the Custom CSS tab.

    The style bundle is version-specific. This means that for each rendering version there is a corresponding version of the style bundle.
